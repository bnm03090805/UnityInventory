## 👨‍🏫 프로젝트 소개
유니티로 인벤토리를 구현하는 프로젝트

## 💜 주요기능

- 기능 1 : UI 전환

- 기능 2 : 현재 스탯을 볼 수 있는 기능

- 기능 3 : 테스트 아이템 생성 버튼을 누르면 아이템이 생성됨

- 기능 4 : 아이템을 생성하는 즉시 인벤토리에 반영

- 기능 5 : 인벤토리이 아이템 아이콘을 클릭하면 장착이 되며 빨간 테두리가 표시되고 장착된 아이템을 클릭 시 해제됨

- 기능 6 : 장착한 아이템의 스탯이 캐릭터의 스탯에 반영됨
